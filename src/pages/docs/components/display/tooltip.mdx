## Examples

### With Arrow Indicator

To add an arrow indicator to the tooltip, add `arrowClassName='fill-theme-muted'` prop to `Tooltip.Content`.

<DocsComponent.Example
  initialCode={`
() => (
<Tooltip.Root>
  <Tooltip.Trigger>
    <Button className='rounded-full' icon={<HiPlus className='w-full h-full' />} />
  </Tooltip.Trigger>
  <Tooltip.Content arrowClassName='fill-theme-muted'>
    Add to library
  </Tooltip.Content>
</Tooltip.Root>
)`}
  scope={{
    Input,
    Button,
    Card,
    Text,
    Tooltip,
    HiPlus
  }}
  imports={{
    "@parssa/universal-ui": {
      named: ["Input", "Button", "Card", "Text", "Tooltip"]
    },
    "react-icons/hi": {
      named: ["HiPlus"]
    }
  }}
/>

You can also enable this globally by setting `tooltip.content_arrow: 'fill-theme-muted'` in your [`UniversalUIConfigProvider`](/docs/getting-started/customization).
