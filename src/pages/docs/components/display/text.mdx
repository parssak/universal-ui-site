## Examples

### Rendering variants as other elements

Sometimes you want the styling of an `h1`, but you want to render it as a `p` or `span`. You can do this by passing the `as` prop.

<DocsComponent.Example
  initialCode={`
() => (
<div>
  <Text variant='h1' as='p'>
  h1 styling as a p
</Text>
<Text variant='h2' as='span'>
  h2 styling as a span
</Text>
</div>
)`}
  scope={{
    Text
  }}
  imports={{
    "@parssa/universal-ui": {
      named: ["Text"]
    }
  }}
/>
