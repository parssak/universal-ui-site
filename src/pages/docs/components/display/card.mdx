## Examples

### Inputs inside

Input elements such as [`Button`](/docs/components/inputs/button) and [`Select`](/docs/components/inputs/select) that render inside cards have slightly elevated borders to make them stand out from the card background.
This can be prevented by adding `data-uui='false'` to the `<Card />` component.

<DocsComponent.Example
  initialCode={`
() => (
<div className='grid md:grid-cols-2 gap-2'>
  <Card className='sm:w-96 md:w-[32rem]'>
    <Card.Content>
      <Text className='mb-size-y' variant="h4">Standard</Text>
      <Text size='sm' className='mt-size-2y'>The borders are a bit more pronounced</Text>
      <InputGroup className='mt-size-2y'>
        <Button theme="brand">Save changes</Button>
        <Select>
          <Select.Trigger />
          <Select.Panel>
            {FRUITS.map((fruit) => (
              <Select.Item value={fruit} key={fruit} className="capitalize">
                {fruit}
              </Select.Item>
            ))}
          </Select.Panel>
        </Select>
      </InputGroup>
    </Card.Content>
  </Card>
  <Card data-uui='false' className='sm:w-96 md:w-[32rem]'>
    <Card.Content>
      <Text className='mb-size-y' variant="h4">With data-uui=false</Text>
      <Text size='sm' className='mt-size-2y'>Borders are more subtle, making it a bit more muddy.</Text>
      <InputGroup className='mt-size-2y'>
        <Button theme="brand">Save changes</Button>
        <Select>
          <Select.Trigger />
          <Select.Panel>
            {FRUITS.map((fruit) => (
              <Select.Item value={fruit} key={fruit} className="capitalize">
                {fruit}
              </Select.Item>
            ))}
          </Select.Panel>
        </Select>
      </InputGroup>
    </Card.Content>
  </Card>
</div>
)`}
  scope={{
    Input,
    Button,
    Card,
    Text,
    Select,
    InputGroup,
    FRUITS
  }}
  imports={{
    "@parssa/universal-ui": {
      named: ["Input", "Button", "Card", "Text", "Select", "InputGroup"]
    }
  }}
/>
