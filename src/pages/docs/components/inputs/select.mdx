## Examples

### With Icons

<DocsComponent.Example
  initialCode={`() => 
      <>
        <Input
          leadingIcon={<Icon.HiBell className="w-full h-full" />}
          placeholder="Notifications" />
        <Input
          trailingIcon={<Icon.HiArrowRight className="w-full h-full" />}
          placeholder="Next" />
      </>
    `}
  scope={{
    Input,
    Icon
  }}
  imports={{
    "@parssa/universal-ui": {
      named: ["Input"]
    },
    "react-icons/hi": {
      default: "Icon"
    }
  }}
/>

You can use the `leadingIcon` and `trailingIcon` props to add icons before/after the Input content.

These props accept any ReactNode, so you can use any icon library you want.
If the icons are not being sized correctly, make sure there is no implicit `width/height` props on the icon, and try adding `w-full h-full`.

### With InputGroup

<DocsComponent.Example
  initialCode={`() =>

          <Input placeholder="First Name" />
          <Input placeholder="Last Name" />

        <Input placeholder="sven@mail.com" type="mail" className="w-full" />
      </div>
    `}
  scope={{
    Input,
    Icon,
    InputGroup
  }}
  imports={{
    "@parssa/universal-ui": {
      named: ["Input", "InputGroup"]
    },
    "react-icons/hi": {
      default: "Icon"
    }
  }}
/>

You can use the `Input` component inside an `InputGroup` to create a segmented Input. The `InputGroup` can be used with `Input`, `Select` and `Input`.

Learn more about `InputGroup` [here ->](/docs/components/inputs/input-group).
