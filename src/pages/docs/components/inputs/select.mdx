## Examples

### With Icons

You can change the trailingIcon of the `Select.Trigger` to change the icon.

<DocsComponent.Example
  initialCode={`() => 
<>
  <Select>
    <Select.Trigger
      placeholder="Notification Type"
      trailingIcon={<Icon.HiBell className="w-full h-full" />}
    />
    <Select.Panel>
      {FRUITS.map((fruit) => (
        <Select.Item value={fruit} key={fruit} className="capitalize">
          {fruit}
        </Select.Item>
      ))}
    </Select.Panel>
  </Select>

  <Select>
    <Select.Trigger trailingIcon={<Icon.HiChevronDown className="w-full h-full" />} />
    <Select.Panel>
      {FRUITS.map((fruit) => (
        <Select.Item value={fruit} key={fruit} className="capitalize">
          {fruit}
        </Select.Item>
      ))}
    </Select.Panel>
  </Select>
</>
`}

scope={{
    Select,
    Icon,
    FRUITS
  }}
imports={{
    "@parssa/universal-ui": {
      named: ["Select"]
    },
    "react-icons/hi": {
      default: "Icon"
    }
  }}
/>

You can use the `leadingIcon` and `trailingIcon` props to add icons

These props accept any ReactNode, so you can use any icon library you want. If the icons are not
being sized correctly, make sure there is no implicit `width/height` props on the icon, and try
adding `w-full h-full`.

### With InputGroup

<DocsComponent.Example
  initialCode={`() => 
<div className="grid gap-2">
  <InputGroup>
    <Input placeholder="First Name" />
    <Select>
      <Select.Trigger placeholder="Role" className="border-theme-base" />
      <Select.Panel>
        <Select.Item value={"Member"} key={"Member"} className="capitalize">
          {"Member"}
        </Select.Item>
        <Select.Item value={"Admin"} key={"Admin"} className="capitalize">
          {"Admin"}
        </Select.Item>
        <Select.Item value={"Owner"} key={"Owner"} className="capitalize">
          {"Owner"}
        </Select.Item>
      </Select.Panel>
    </Select>
  </InputGroup>
  <Input placeholder="sven@mail.com" type="email" className="w-full" />
</div>
`}
  scope={{
    Input,
    Icon,
    InputGroup,
    Select
  }}
  imports={{
    "@parssa/universal-ui": {
      named: ["Input", "InputGroup", "Select"]
    }
  }}
/>

You can use the `Select` component inside an `InputGroup` to create joint forms. The `InputGroup` can be used with `Input`, `Select` and `Input`.

Learn more about `InputGroup` [here ->](/docs/components/inputs/input-group).
